Custom Labels Examples

1. Default (Icon-Only Rotate/Flip)

By default, rotate and flip buttons show only icons.

2. With Labels for Rotate and Flip

Add custom labels to rotate and flip buttons.

3. Declarative Approach with Custom Labels

Using data attributes to set custom labels.

4. Internationalization (Bahasa Melayu)

All buttons with Malay labels.

5. Mixed Labels

Main actions have labels, rotate/flip remain icon-only.

Code Examples

Programmatic Approach:

// Default (icon-only)
new SWP(container, {
  // rotateLeft, rotateRight, flipH, flipV are null by default
});

// With labels
new SWP(container, {
  labels: {
    rotateLeft: 'Rotate Left',
    rotateRight: 'Rotate Right',
    flipH: 'Flip Horizontal',
    flipV: 'Flip Vertical'
  }
});

Declarative Approach:

<div
  data-swp
  data-swp-labels="rotateLeft: 'Rotate Left'; rotateRight: 'Rotate Right'; flipH: 'Flip H'; flipV: 'Flip V'"
></div>
