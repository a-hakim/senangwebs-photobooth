SWP Declarative Approach

Configure photo editor using HTML data attributes - no JavaScript required!

1. Basic Declarative Initialization
Simply add data-swp attribute to auto-initialize
<div data-swp></div>
2. Custom Dimensions
Set width and height using data attributes
<div data-swp data-swp-width="900" data-swp-height="500"></div>
3. Icons Only (Compact Mode)
Hide all labels for a compact toolbar
<div data-swp data-swp-show-labels="false"></div>
4. Labels Only (Text Mode)
Hide all icons for text-only buttons
<div data-swp data-swp-show-icons="false"></div>
5. Custom Labels - Simple Format
Customize button labels using key:value pairs separated by semicolons
<div data-swp data-swp-labels="upload: 'Muat Naik'; resize: 'Tukar Saiz'; adjust: 'Laraskan'; filters: 'Penapis'; reset: 'Set Semula'; save: 'Simpan'"></div>
6. Custom Labels - JSON Format
Use JSON format for more complex configurations

7. Hide Specific Labels
Set labels to 'null' to hide only specific button labels
<div data-swp data-swp-labels="rotateLeft: null; rotateRight: null; flipH: null; flipV: null"></div>

üìö Data Attributes Reference
Basic Configuration:
  • data-swp - Enable auto-initialization
  • data-swp-width - Canvas width (number)
  • data-swp-height - Canvas height (number)
  • data-swp-image-url - Initial image URL
UI Configuration:
  • data-swp-show-icons - Show/hide icons (true/false)
  • data-swp-show-labels - Show/hide labels (true/false)
  • data-swp-labels - Custom labels (see formats below)
Label Format Options:
Simple Format: data-swp-labels="upload: 'Text'; resize: 'Text'; save: 'Text'"
JSON Format: data-swp-labels='{"upload":"Text","resize":"Text","save":"Text"}'
Available Label Keys: upload, rotateLeft, rotateRight, flipH, flipV, resize, adjust, filters, reset, save
